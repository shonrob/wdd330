The Window Object

The Browers Object Model (BOM) is a collection of properties and methos that contain information about the browser and computer screen.

The DOM is a programming interface (API) for representing and interacting with HTML, XHTML, and XML documents.

    BOM
  • window object- navigator
  • window object- userAgent
  • window location.href, .protocol, .hostname, .pathname, .search. .orgin
  • controling window: .open(), .close(), .moveTo(), .resizeTo(), .screen: .height or .width / .availWidth or availHeight, .colorDepth

HTML5 APIs

    data- Attribute
  • A way of embedding data in a web page using custom attributes that are ignored by the browser.
  • Private to the page-sole purpose is to serve the JavaScript program
    Web Storage
  • provides a key-value store on the client's computer that is similar to using cookies but has few restriciotns, more storage capacity and is generally easier to use.
  • window.localStorage stores data
  • window.sessionStorage holds the info for only the session.
    • key tells us the key of the item that changed
    • newValue tells us the new value to which it has been changed
    • oldValue tells us the previous value before it was changed
    • storageArea tells us if it is sotred in local or session storage
